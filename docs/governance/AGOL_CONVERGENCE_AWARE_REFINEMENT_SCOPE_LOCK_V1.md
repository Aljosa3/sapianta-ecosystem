# AGOL Convergence-Aware Refinement Scope Lock V1

Status: finalized scope lock declaration.

Primitive:
`AGOL_CONVERGENCE_AWARE_REFINEMENT_V1`

## Locked Scope

The primitive is strictly limited to:

- convergence detection;
- stabilization recommendation;
- continuity protection guidance;
- local refinement recommendation;
- freeze-zone recommendation;
- mutation pressure awareness;
- replay-visible refinement stabilization.

## Locked Non-Execution Rule

The primitive does not execute commands, mutate files, start runtime processes,
deploy software, freeze UI files, block user mutations, or create orchestration
authority.

The primitive may generate guidance only.

## Locked Proposal-Only Rule

The primitive may:

- detect convergence signals;
- recommend stabilization;
- identify stabilized Product 1 regions;
- recommend continuity-protected regions;
- recommend local-only refinement;
- warn about mutation pressure risk;
- preserve replay-visible convergence evidence.

The primitive may not:

- redesign Product 1 autonomously;
- automatically freeze UI;
- automatically block mutation;
- mutate Product 1 automatically;
- execute runtime actions;
- create runtime authority;
- create orchestration authority;
- create deployment authority;
- bypass governance continuity;
- override human approval.

## Controlled Changes

The following require governed review and renewed approval:

- adding automatic Product 1 mutation;
- adding automatic freeze enforcement;
- adding mutation blocking behavior;
- adding runtime execution;
- adding browser, server, or deployment orchestration;
- adding autonomous redesign loops;
- changing the user final authority requirement;
- changing the replay identity fields;
- expanding the primitive beyond proposal-only convergence guidance.

## Non-Inheritance Rule

Certification of this primitive does not authorize:

- UI generation;
- runtime mutation;
- autonomous product redesign;
- deployment automation;
- automatic file freezing;
- self-directed execution;
- refusal of user-directed future refinements.

All future use remains bounded by explicit user-directed task scope.
