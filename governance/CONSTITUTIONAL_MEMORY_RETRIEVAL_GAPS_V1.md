# Constitutional Memory Retrieval Gaps V1

Status: retrieval model gap analysis.

## Critical Gaps

Critical gaps before implementation:

- no retrieval request schema
- no citation bundle schema
- no replay-visible consultation record schema
- no source excerpt boundary
- no automated source classification validation
- no implementation proof that provider and worker triggers are impossible

## Important Gaps

Important gaps:

- canonical source inventory remains static and manual
- JSON artifact schema variation remains across epochs
- `.runtime/` evidence linkage remains conditional
- no operator-facing retrieval command exists
- no governance review workflow for conflict status exists

## Optional Gaps

Optional gaps:

- citation examples
- consultation report template
- current-baseline digest
- artifact-to-layer lookup table
- source freshness report

## Non-Gaps

Not required:

- semantic search
- vector memory
- memory cache
- autonomous retrieval
- correction loop implementation
- provider memory access
- worker memory access
