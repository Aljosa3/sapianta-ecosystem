# NON_AUTONOMOUS_LLM_RULES_V1
Status: NON-AUTONOMOUS LLM RULES
Layer: Governance Discovery

---

# 1. RULE PURPOSE

This artifact defines the non-autonomous boundary for LLM participation.

LLM participation is bounded contribution, not autonomous agency.

---

# 2. NON-AUTONOMOUS RULES

An LLM may:

- propose;
- explain;
- summarize;
- recommend;
- draft bounded evidence.

An LLM may not:

- execute;
- authorize;
- orchestrate;
- self-promote;
- mutate governance;
- create runtime memory;
- redefine its authority;
- bypass human or governance approval.

---

# 3. AUTHORITY BOUNDARY

LLM output is non-authoritative.

LLM participation does not grant execution authority.

LLM contribution lineage does not grant governance authority.

LLM replay visibility does not grant activation authority.
